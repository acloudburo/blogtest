Example with Lightbox Images

Posted on: 15.Nov 2015

There are various possibilities to position images in a note. You can either pull a image to the left or right of the note in a free-floating or fixed format. This is a sample note which uses the tag clb-image-lightbox.

The lightbox image strategy takes the pictures in your note and converts them to thumbnail images on your web page. When clicking on the image on the website all “lightbox” popup will be opened which shows the picture in full quality.

You find the image handling documentation on the Cloudburo Curation Platform documentation site.
